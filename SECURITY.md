# Security Policy

## Supported Versions

Only the latest version of Lastik is supported for security updates.

## Local-Only Promise

Lastik is designed to be run entirely in the browser. It does not send any text data to any remote server. The only outgoing requests are to load standard web fonts.

## Reporting a Vulnerability

If you discover a security vulnerability, please open an issue on GitHub with the label `security` or contact the maintainer directly.

We take security seriously and will investigate all reports promptly.
